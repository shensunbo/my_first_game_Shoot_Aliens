# my_first_game_Shoot_Aliens
Vertical shooter built with Python + pygame. Current build includes player movement, bullets with cooldown, basic enemy waves, collisions, score, and lives with restartable game over screen.

## Run
```bash
python3 shoot_aliens_py/day0.py
```

## Controls
- Arrow keys: move (up/down/left/right)
- Space or **F**: fire
- R: restart after game over
- Q or Esc: quit from game over screen
